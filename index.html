    Intro

    ArduinoPhone is a Mobile Phone make with Arduino(or Seeeduino) and some other modules.

    ArduinoPhone features:

    • Phone call
    • SMS
    • Contacts form
    • Clock
    • Gesture for navigate UI
    • Touch input

    Before you uplaod source code to Arduino, you should copy all contents in Libraries Directory into ..arduino-1.0.4\libraries.

    If you want to know how build it step by step, look at this.

    Modules: 1.Arduino Uno 2.TFT Touch Shield 3.GPRS Shield 4.RTC 5.Custom ArduinoPhone Charge Circuit (or Lipo Rider) 6.Li-po battery 7.[A shell (with 3D printer)]
